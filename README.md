Launcher for rust arcade cabinet

# How to

Put executable folders in ./apps/ (it's gitignored)

You can copy test_apps to apps for a minimal test.

example hierarchy: 

<pre>
./apps
--- game1/
    --- game1.sh
--- game2_using_bevy/
    --- executable_name
    --- assets/
        --- images.png
...
</pre>
